<h2 align="center"> Como rodar o Projeto </h2>

<h3 align="center">🔧 Instalação</h3>

```
git clone https://github.com/ViKMacedo/To-do.git
```
recomendo que instale as dependências também

```
cd To-do npm install
```
e pode dar run com o 

```
npm start
```
